# 🎭 lobster-persona — M1 帳號定位模組

> **裝完這個，你的每個帳號都有完整人設，不再是空殼帳號**

## ❓ 你遇到的問題

- 帳號註冊完不知道怎麼定位
- 大頭照用真人怕被肉搜，用假的又假到不行
- 封面圖、自我介紹想了半天還是很假
- 養了 10 個帳號，每個長得一模一樣

## ✅ 裝了這個模組之後

1. 告訴龍蝦你的目的（例：房仲開發、美業推廣、團購主）
2. AI 自動反推：年齡、性別、風格、興趣
3. 自動生成：大頭照（AI 二創，不盜圖）+ 封面圖 + 自我介紹
4. 搭配 AdsPower，批量為每個帳號設定不同人設
5. 每個帳號都像真人，降低被封風險

## 🔧 技術架構

```
你的目的 → AI 逆向規劃人設
         → AutoIP / Manus 生成圖片
         → AdsPower 批量套用
         → 每個帳號獨一無二
```
